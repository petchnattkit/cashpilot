# Task: Implement SKU Widgets

## User Story
**As** a Business Owner,
**I want** to know which products are selling the most (Value vs Volume),
**So I can** manage my inventory levels.

## Atomic Scope
- Update `EntityWidgets.tsx`.
- Implement "Top SKUs" widget (List or Chart).
- Show metrics: Total Value Sold, Frequency.

## Acceptance Criteria
1. Navigate to `/dashboard`.
2. Verify "Top Products" widget appears.
3. Check calculation (Value = sum of amounts).

## Impact Area
- Frontend Route `/dashboard`
