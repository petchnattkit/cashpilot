# Task: Update Transaction Form (SKU Selection)

## User Story
**As** a User,
**I want** to select a specific product (SKU) when recording a transaction,
**So I can** track sales performance per product.

## Atomic Scope
- Update `TransactionForm` in `TransactionsPage.tsx`.
- Replace free-text `sku` input with a Dropdown (Select).
- Populate dropdown using `useSkus`.
- (Optional) "Add New" button to open SKU creation modal.

## Acceptance Criteria
1. Open "Add Transaction" modal.
2. Click SKU dropdown. See list of SKUs.
3. Select SKU.
4. Verify `sku_id` is included in payload.

## Impact Area
- Frontend Route `/transactions`
