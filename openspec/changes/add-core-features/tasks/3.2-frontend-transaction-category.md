# Task: Update Transaction Form (Category Selection)

## User Story
**As** a User,
**I want** to categorize my transactions (e.g., "Office Supply", "Sales"),
**So I can** analyze my cashflow sources and sinks.

## Atomic Scope
- Update `TransactionForm` in `TransactionsPage.tsx`.
- Replace free-text `category` input with a Dropdown (Select).
- Populate dropdown using `useCategories`.

## Acceptance Criteria
1. Open "Add Transaction" modal.
2. Click Category dropdown. See list.
3. Select Category.
4. Verify `category_id` is included in payload.

## Impact Area
- Frontend Route `/transactions`
