# Task: Implement Settings Page

## User Story
**As** a Business Owner,
**I want** to configure my baseline cash value,
**So I can** override the calculated runway in the dashboard.

## Atomic Scope
- Create `src/pages/SettingsPage.tsx`.
- Implement local storage persistence for "Baseline Cash".
- Form with Input field and Save button.

## Acceptance Criteria
1. Navigate to `/settings`.
2. Enter value "10000", Click Save.
3. Reload page. Value "10000" persists.
4. **Constraint**: Simple local storage interaction, no DB needed for this task.

## Impact Area
- Frontend Route `/settings`
