# Task: Update TypeScript Interfaces

## User Story
**As** a Developer,
**I want** my TypeScript interfaces to match the database schema,
**So I can** write type-safe code for the new features.

## Atomic Scope
- Update `src/types/database.ts` (or equivalent).
- Add `Sku` and `Category` interfaces.
- Update `Transaction` interface to include `sku_id` and `category_id`.

## Acceptance Criteria
1. Run `npm run typecheck` (tsc).
2. Verify code compiles without type errors related to new fields.
3. **Constraint**: Must match the SQL schema defined in 1.1 and 1.2.

## Impact Area
- Frontend Type System
