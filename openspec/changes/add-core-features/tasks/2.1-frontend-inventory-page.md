# Task: Implement Inventory Page

## User Story
**As** a Business Owner,
**I want** to see a list of my products (SKUs),
**So I can** check what I have in my inventory.

## Atomic Scope
- Create `src/pages/InventoryPage.tsx`.
- Implement `useSkus` hook (read from `skus` table).
- Display a table with columns: Code, Name, Description.
- Add "Create SKU" button and Modal form.

## Acceptance Criteria
1. Navigate to `/inventory`.
2. Verify list renders (empty state or populated).
3. Click "Add SKU", fill form, save.
4. Verify new SKU appears in list.
5. **Constraint**: Use shared UI components (`DataTable`, `Modal`).

## Impact Area
- Frontend Route `/inventory`
