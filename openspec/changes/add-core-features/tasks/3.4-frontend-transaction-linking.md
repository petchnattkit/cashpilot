# Task: Transaction Entity Linking

## User Story
**As** a User,
**I want** to link income to Customers and expenses to Suppliers,
**So I can** track who owes me money or who I am paying.

## Atomic Scope
- Update `TransactionForm` in `TransactionsPage.tsx`.
- IF `cash_in` > 0: Show "Customer" dropdown (Required).
- IF `cash_out` > 0: Show "Supplier" dropdown (Required).
- Populate using `useCustomers` and `useSuppliers` hooks.

## Acceptance Criteria
1. Enter `cash_in` = 100. Verify User MUST select a Customer.
2. Enter `cash_out` = 50. Verify User MUST select a Supplier.
3. Verify `customer_id` or `supplier_id` is sent to DB.

## Impact Area
- Frontend Route `/transactions`
