## ADDED Requirements
### Requirement: Configuration Settings
The system SHALL allow users to configure global application settings.

#### Scenario: Baseline Value Configuration
- **WHEN** the user accesses the Settings page
- **THEN** they SHALL be able to set a "Baseline Value" for graphs
- **AND** this value SHALL be used to visually determine if business metrics are above or below the baseline

### Requirement: Master Data Management
The system SHALL allow management of core reference data.

#### Scenario: Transaction Categories
- **WHEN** the user accesses the Master Data section in Settings
- **THEN** they SHALL be able to view, add, and edit Transaction Categories used in the Transaction page
